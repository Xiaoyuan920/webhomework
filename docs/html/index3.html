  • 经济学院开展第四十六期学术沙龙

    发布者： 时间：2022-11-20 14:47:47 浏览：18

    11月9日下午，经济学院在义丰楼619开展了第四十六期学术沙龙活动，本期沙龙由金融学教研室张书君博士主讲，沙龙主题为“中国股票市场有效性及监管研究”。金融学教研室全体教师、学院博士以及部分有兴趣的老师参加了此次沙龙活动，活动由金融教研室主任李拔豪主持。

    张书君的讲座内容来源于其前期的研究成果。主要从中国股票市场的有效性与信息有效性、有效市场假说和随机游走假设、实证检验结果及原因分析、基于有效性的金融博弈、股市有效性战略的制定等方面作了详细的介绍。讲座对金融学专业教师教学研究带来了启发，与会教师基于讲座内容展开了讨论与交流，对张博士选题的研究方法、研究内容提出了建议和思考。
